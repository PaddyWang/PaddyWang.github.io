【CSS】常用选择器

By Paddy

发表于 2016-03-05

基本选择器

  • 类选择器
    • .class
  • id 选择器
    • #id
  • 标签选择器
    • div
  • 通配符
    • *

关系选择器

  • 并集选择器
    • div, p, ul, li

  • 后代选择器

    • ul li
    • 选择<ul>元素内的所有<li>元素

  • 子代选择器

    • ul > li
    • 选择在<ul>元素内的第一层<li>元素

  • 相邻兄弟选择器

    • div + p
    • 选择紧接在<div>后面第一个<p>元素

  • 一般兄弟选择器

    • div ~ p
    • 选择与<div>同父元素的且在<div>元素之后的所有兄弟<p>元素

属性选择器

  • div[id]

    • 选择带有id属性的所有<div>元素

  • a[target="_blank"]

    • 选择target=_blank的所有<a>元素

  • div[class|=box]

    • 选择class值为box的或者以box-开头的所有<div>元素

  • div[title~="hello"]

    • 选择title值为hello完整单词的或者是包含hello完整单词的所有<div>元素

  • a[href^="http"]

    • 选择href值以http开头的所有的<a>元素

  • img[scr$=".png"]

    • 选择src值以.png结尾的所有<img>元素

  • p[class*="text"]

    • 选择class值中包含text的所有<p>元素

伪元素选择器

  • p::first-letter
    • 选择所有<p>元素的第一个字符
  • p::first-line
    • 选择所有<p>元素的第一行字符
  • div::before
    • 在每个div之前插入的内容(content)及样式
  • div::after
    • 在每个div之后插入的内容(content)及样式
  • p::selection
    • 选择所有<p>元素被用户选取的内容样式
    • notice：只能设置color、background、cursor 以及 outline

伪类选择器

选择该父元素内的元素

  • h1:first-child

    • 选择h1的父元素下的第一个标签，并且为h1的<h1>元素

  • h1:last-child

    • 选择h1父元素下的最后一个标签，并且为h1的<h1>元素

  • h3:first-of-type

    • 选择h3父元素下第一个出现的<h3>元素

  • h3:last-of-type

    • 选择h3父元素下最后一个出现的<h3>元素

  • h3:only-of-type

    • 选择h3父元素下仅仅只有一个<h3>元素（可以含有其他标签）的<h3>元素

  • h3:only-child

    • 选择没有兄弟元素的<h3>元素

  • p:nth-child(3)

    • 选择p父元素下第三个元素是p元素的<p>元素
    • 参数支持：an+b 形式 即2n+1
    • 其中a可正可负，b必须为正
    • 当a为负时，-n+3 选择倒数三个
    • 下例：p:nth-child(2n+1)

  • p:nth-last-child(3)

    • 选择p父元素下倒数第三个是p元素的<p>元素

  • p:nth-of-type(3)

    • 选择p父元素下第三个<p>元素

  • p:nth-last-of-type(3)

    • 选择p父元素下倒数第三个<p>元素

  • a:link

    • 未被访问的a链接
  • a:visited
    • 已被访问的a链接
  • a:active
    • a链接被激活状态
  • a:hover
    • 鼠标经过a链接
  • input:focus
    • 选择获取焦点的<input>元素
  • :root
    • 选择文档根元素即<html>元素
  • div:empty
    • 选择空的div元素（也不可以有文本）
  • p:target
    • 选择当前活动的<p>元素
    • (一般用于锚链接)
  • input:enabled
    • 选择激活的<input>元素
  • input:disabled
    • 选择禁用的<input>元素
  • :not(p)
    • 选择非p的元素
    • notice：p要设置相应的样式，否则也会调用:not(p)里的样式
